# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text is at the URL above. The summary: be respectful, assume
good faith, and treat the comment threads on this repository the way you
would treat a code review with a colleague you respect.

## Reporting

If you observe behavior that violates the Code of Conduct, open a GitHub
issue using the **Bug Report** template and apply the `conduct` label,
or contact the repository maintainers directly. Reports are handled
confidentially.

## Scope

This Code of Conduct applies to all project spaces — issues, pull
requests, discussions — and to behavior outside those spaces when an
individual is representing the project.
